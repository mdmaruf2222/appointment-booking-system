# Online Appointment Booking System

A web-based appointment booking system developed using PHP and MySQL.

## Features
- Book appointments online
- Admin panel to manage bookings
- CRUD operations
- MySQL database integration

## Technologies
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## How to Run
1. Install XAMPP
2. Start Apache & MySQL
3. Place project in htdocs folder
4. Create database `appointment_db`
5. Create table `appointments`
6. Open `http://localhost/appointment_system/index.php`
